Preface

In this article I will show you the basics of how to use the programming language C. If you have not done any programming before, I would recommend you learn a programming language like Python or Java instead, as they are much friendlier to beginner programmers.

Installing a Toolchain

Before we can begin, we need to install what's called a toolchain. The toolchain is what turns our program's source code into something the computer can execute. Below are 3 different guides for installing a toolchain on different operating systems.

Windows

On Windows there are a few existing toolchains such as MinGW and MSVC, although for this guide I will show you how to install Fabrice Bellard's Tiny C Compiler. To install it, go to Fabrice Bellard's website , download the tcc-0.9.27-win64-bin.zip file and extract its contents to a folder somewhere.

MacOS

MacOS luckily has a built-in development suite called XCode, which includes the required tools for C, C++ and Objective-C programming. To install it, you can either download XCode from the App Store or open Terminal and type cc. This will show a pop-up window asking if you would like to install XCode. Click yes and MacOS will automatically download and install the tools we need.

Linux

Most Linux distributions have a built-in package manager that can install the packages we need. Below are the commands required to install GCC with various different Linux distributions' package managers.

Debian / Ubuntu

# apt install build-essential

Arch / Manjaro

# pacman -S base-devel

Alpine

# apk add build-base

First Program

The following code block is the smallest possible C program that will compile and run. 
int main() {
    return 0;
}

Now, this program doesn't actually do anything. Let's write a program that writes out text. 
#include <stdio.h>

int main() {
    puts("Hello World!");
    return 0;
}
